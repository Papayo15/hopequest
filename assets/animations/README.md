# Lottie Animations

Este directorio debe contener 16 animaciones Lottie (JSON).

## Lista de Animaciones Requeridas (16 archivos JSON)

### Portales (6 animaciones)
- `portal_aereo.json` - Avión despegando
- `portal_maritimo.json` - Barco navegando
- `portal_terrestre.json` - Bus/carro en carretera
- `portal_clandestino.json` - Puerta oscura con sombras
- `portal_refugiado.json` - Tienda de campaña humanitaria
- `portal_familiar.json` - Familia reuniéndose

### Collection Effects (2 animaciones)
- `collect_star.json` - Estrella brillando al recolectar
- `collect_coin.json` - Moneda girando

### Celebrations (3 animaciones)
- `celebration_country_complete.json` - Confeti y celebración
- `celebration_level_up.json` - Subida de nivel con rayos
- `celebration_achievement.json` - Trofeo apareciendo

### UI Feedback (3 animaciones)
- `loading_spinner.json` - Spinner de carga personalizado
- `success_checkmark.json` - Check verde apareciendo
- `error_x.json` - X roja apareciendo

### Character Effects (2 animaciones)
- `character_walk.json` - Personaje caminando (loop)
- `character_jump.json` - Personaje saltando

## Especificaciones Técnicas

- **Formato**: JSON (Lottie)
- **Tamaño máximo**: 100KB por animación
- **FPS**: 30fps
- **Duración**:
  - Loops: 1-2 segundos
  - One-shots: 1-3 segundos
- **Optimización**: Comprimido, sin assets externos

## Cómo Crear

### Opción 1: LottieFiles Creator (Gratis)
1. Ve a https://lottiefiles.com/creator
2. Crea animación simple
3. Exporta como JSON
4. Optimiza con https://lottiefiles.com/tools

### Opción 2: After Effects + Bodymovin
1. Diseña en After Effects
2. Exporta con plugin Bodymovin
3. Optimiza tamaño

### Opción 3: LottieFiles Marketplace (Buscar gratuitas)
1. https://lottiefiles.com/featured
2. Busca animaciones similares gratuitas
3. Modifica colores si es necesario
